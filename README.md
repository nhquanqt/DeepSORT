# DeepSORT
## Preparing
Create these folders
```
|DeepSORT
|---|datasets
|---|---|cam_xx.json
|---|---|cam_xx.mp4
|---|---|cam_xx.txt
|---|model_data
|---|---|mars-small128.pb
```
The `mars-small128.pb` file is downloaded from [here](https://drive.google.com/drive/folders/1m2ebLHB2JThZC8vWGDYEKGsevLssSkjo)
